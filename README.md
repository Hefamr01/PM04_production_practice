# PM04_production_practice – Производственная практика (ПМ.04)

## 📌 О репозитории

Этот репозиторий содержит материалы производственной практики по профессиональному модулю **ПМ.04 «Сопровождение и обслуживание программного обеспечения компьютерных систем»**.

Практика проходила в **Управлении социального развития администрации Слободского района** (г. Слободской, Кировская область).

**Сроки практики:** 27.05.2026 – 16.06.2026

**Студент:** Мурин Денис Валерьевич, группа 23П-1, 3 курс

## 📁 Содержание репозитория

- **Документация/** – отчёт, дневник, аттестационный лист, презентация
- **Скриншоты/** – скриншоты всех этапов (установка ПО, настройка AD, права доступа, тестирование)

## 🛠 Краткое описание выполненного

- Установка и настройка веб-сервера Laragon, CMS Moodle и WordPress, 1С:Предприятие, антивируса Kaspersky Free, дополнительного ПО.
- Сборка Notepad++ из исходного кода.
- Развёртывание Active Directory (роль «Доменные службы Active Directory», контроллер домена).
- Настройка локальных и доменных политик безопасности.
- Разграничение прав доступа для пользователей «Администратор» и «Оператор».
- Тестирование работоспособности и совместимости.
- Сохранение конфигураций в Git.
